# Colinch AI 경량 하네스

- 공통 규칙은 상위 `AGENTS.md`를 따른다. 자동 포스팅 중 `CLAUDE.md` 전체를 읽지 않는다.
- 언어: 한국어 직접체 `~다`. 대상은 AI 도구가 궁금한 직장인·학생·일반인이다.
- 주제: AI 도구 사용법, 상황별 프롬프트, AI 부업 현실, AI 뉴스 해설, 도구 비교·추천.
- 글의 중심은 “그래서 실제로 어디에 쓰나”다. 장점만 쓰지 말고 아쉬운 점과 실패 상황도 넣는다.
- 웹 검색: 최신 AI 뉴스·도구 변화가 핵심이면 1~2회만 확인한다. 제품명·날짜·출처를 본문에 구체적으로 쓴다.
- SEO: 도구명(`ChatGPT`, `Claude`, `Gemini`, `Perplexity`, `Copilot` 등)과 사용 상황을 title·description·첫 문단에 넣는다.
- 태그 후보: `ChatGPT`, `Claude`, `Gemini`, `Perplexity`, `Copilot`, `AI글쓰기`, `AI이미지`, `AI검색`, `AI요약`, `AI부업`, `프롬프트`, `업무자동화`, `AI뉴스`, `OpenAI`, `Anthropic`, `GoogleAI`.
- 시각 자료: 도구별 비교표, 작업 전후 예시 표, 프롬프트 구조 표, 기능 변화 타임라인을 우선한다.
- 복사 가능한 프롬프트를 넣을 때는 바로 쓸 수 있게 짧고 구체적으로 작성한다.
