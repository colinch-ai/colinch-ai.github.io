ChatGPT가 드디어 직접 움직이기 시작했다. 답을 알려주는 게 아니라, 실제로 행동한다.

## 왜 지금 이게 화제인가

기존 ChatGPT는 "경쟁사 조사해줘"라고 해도 결국 텍스트로 끝났다. 뭔가 더 하려면 내가 직접 창을 열고, 사이트에 들어가서, 클릭해야 했다.

ChatGPT Agent는 다르다. OpenAI가 최근 정식 출시한 이 기능은 ChatGPT가 직접 웹사이트를 탐색하고, 정보를 수집하고, 파일까지 만들어낸다. Pro·Plus·Team·Enterprise·Edu 플랜이면 지금 쓸 수 있다.

기존에 Operator라는 별도 서비스가 있었는데, 이게 ChatGPT에 완전히 통합됐다. `operator.chatgpt.com`은 곧 문 닫는다. Operator 쓰던 사람이라면 이제 ChatGPT 하나로 다 된다.

## 실제로 뭘 시킬 수 있나

OpenAI가 직접 시연한 예시들이다:

- "내 캘린더 보고, 이번 주 고객 미팅 전에 알아야 할 뉴스 정리해줘"
- "일본식 아침밥 4인분 재료 온라인에서 사줘"
- "경쟁사 세 곳 조사해서 슬라이드 덱 만들어줘"

처음엔 과장 마케팅인 줄 알았다. 그런데 실제로 써보면 ChatGPT가 브라우저를 열고 사이트를 돌아다니는 게 화면에 그대로 보인다. 탭을 옮기고, 검색창에 뭔가 입력하고, 결과를 읽어서 정리한다.

**지금 바로 써볼 수 있는 프롬프트:**

```
"[주제] 관련 최신 뉴스 3개 찾아서, 각각 요점 2줄로 정리한 표 만들어줘"
```

```
"내 이메일에서 이번 주 미팅 일정 찾아서 준비할 항목 체크리스트 만들어줘"
```

```
"[경쟁사 이름] 홈페이지 가서 주요 기능·가격 정리해줘"
```

## 기존 기능들과 뭐가 다른가

![ChatGPT Agent vs 기존 기능 비교](https://images.unsplash.com/photo-1485827404703-89b55fcc595e?w=800&q=80)

이게 헷갈리는 사람이 많아서 정리해둔다.

| 기능 | 하는 것 |
|------|---------|
| 기존 ChatGPT | 텍스트 응답만 |
| 딥리서치 | 인터넷 검색 후 긴 보고서 |
| Operator | 웹사이트 직접 조작 |
| **ChatGPT Agent** | 세 가지 전부 통합 |

Agent는 딥리서치 + Operator + ChatGPT를 하나로 묶은 거다. 조사하고, 웹사이트를 직접 조작하고, 대화로 진행 상황을 알려주는 것까지 한 번에 된다. 뭐가 어디 있는지 기억 안 해도 된다.

## 이런 점은 아직 아쉬웠다

속도가 생각보다 느리다. 웹을 직접 돌아다니다 보니 단순 텍스트 응답보다 몇 배 오래 걸린다. 2~3분 기다리는 건 기본이고, 복잡한 작업은 더 걸린다.

사이트마다 막히는 경우도 있다. 로그인이 필요하거나 봇 차단이 걸린 사이트는 진입 자체가 안 된다. "네이버 로그인해서 메일 확인해줘"는 현재 구조상 어렵다.

"쇼핑 대신 해줘"는 국내에서 불분명하다. 미국 사용자 기준으로 결제까지 시연됐는데, 한국 쇼핑몰에서 카드 결제까지 실제로 되는지는 아직 확인 중이다. 일단은 "조사해서 목록 뽑아줘"까지만 믿는 게 현실적이다.

## 한 줄 정리

ChatGPT가 드디어 "직접 해주는 AI"가 됐다. Plus 플랜이면 지금 바로 써볼 수 있고, Operator 없어지는 김에 한 번 테스트해볼 타이밍이다.
